# アプリケーションを作成した背景
最近ではクレジットカード、QRコード決済、ICカード決済など複数の決済方法が存在しています。従来の家計簿では支出額のみ記入して支出を管理するものが多いため、各決済方法ごとの支出額を把握することができないと感じていました。同様の問題を抱えている人も多いのではないかと推測し、支出入力時に決済方法を選択することで決済ごとの支出額が確認できるような家計簿アプリケーションを開発することにしました。
